### Clonar repositorio

```bash

```bash
git add (nome do arquivo)

```bash
git add .

### Para enviar / commit o código
```bash
git commit -m "Mensagem do commit"

### Para ver o log
```bash
git log

```bash
git status


### Para enviar as alterções para o repositorio remoto
```bash
git push
